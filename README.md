# Global AI Workforce Analytics Dashboard

## Project Overview

This project analyzes the global AI and Data Science job market using Python, Pandas, Matplotlib, and Power BI. The dashboard provides insights into salary trends, experience levels, remote work patterns, and hiring locations across the AI industry.

## Objectives

* Analyze AI job market trends
* Identify top-paying AI job roles
* Compare salaries across experience levels
* Explore remote work opportunities
* Visualize hiring distribution by country

## Tools Used

* Python
* Pandas
* Matplotlib
* Power BI
* CSV Dataset

## Key Insights

* Average Salary: $121,992
* Total Jobs Analyzed: 15,000
* Countries Covered: 20
* Remote Jobs: 33.56%

## Dashboard Features

* KPI Cards

  * Average Salary
  * Remote Work Score
  * Benefits Score

* Visualizations

  * Top Paying AI Job Roles
  * Average Salary by Experience Level
  * Top Hiring Countries
